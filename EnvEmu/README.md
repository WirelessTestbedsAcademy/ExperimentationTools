Always:
-----------

check/guess which /dev/ttyUSBx device is the EE
execute:
```
sudo ln -fs /dev/ttyUSBX /dev/ee
```

---------

```
./tos-bsl --invert-reset --swap-reset-test -c /dev/ee -r -e -I -p main.ihex
```
